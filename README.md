## Installation

OS X & Linux:

```sh
javac Graph.java
compile and class to Built in package
```

Windows:

```sh
javah Graph.java
compile and class to Built in package
```

## Usage example

import java.util.Graph;//import

Graph.addVertex()
Graph.addEdge()

Graph.traverseBFS();
